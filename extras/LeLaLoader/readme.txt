LeLaLoader
10/2008 by Thomas Nesges (http://www.thomasnesges.de)

Installation:
  German:
    1.) Lege ein Verzeichnis im "scripts" Verzeichnis deiner XBMC Installation an
        Der Name des Verzeichnisses kann zB "LeLaLoader" lauten (muss aber nicht)
    2.) Entpacke diese Zip-Datei in dieses Verzeichnis

  English:
    1.) Create a directory in the "scripts" directory of your XBMC installation
        The dictiory may be named "LeLaLoader" (but doesn't has to)
    2.) Unzip this Zip-File there
